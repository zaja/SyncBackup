SyncBackup v1.2 - Korisnički priručnik

Autor: Goran Zajec | Web: https://svejedobro.hr

📖 Uvod

SyncBackup je aplikacija za automatsko backup-iranje i sinkronizaciju foldera. Podržava dva tipa job-ova, automatsko planiranje, SQLite bazu podataka i system tray funkcionalnost.

Glavne značajke:
  • Dashboard s pregledom statistika
  • Simple i Incremental job-ovi
  • Automatsko planiranje izvršavanja
  • ZIP kompresija backup-ova
  • Filtriranje datoteka (exclude patterns)
  • Desktop notifikacije
  • System tray podrška

💡 Zašto SyncBackup?

SyncBackup je nastao iz osobne potrebe nakon što sam nekoliko puta izgubio dane ili tjedne rada na projektima. Najdramatičniji slučaj bio je kada sam radio na WordPress pluginu više od tjedan dana, i tokom testiranja uninstall funkcije slučajno obrisao cijeli project folder. U tom trenutku mi je postalo kristalno jasno - ovo se više ne smije dogoditi.

Potražio sam postojeća rješenja za backup i sinkronizaciju, ali:

  • Većina je preteška i preglomazna za moje potrebe
  • Spore su i troše previše resursa
  • Često nisu intuitivne za korištenje
  • Nisu fleksibilne za različite scenarije backup-a

Rješenje? Napraviti vlastitu aplikaciju koja je:

  • ✅ Brza i lagana - bez nepotrebnih funkcija
  • ✅ Jednostavna - intuitivno sučelje
  • ✅ Fleksibilna - Simple i Incremental backup-ovi
  • ✅ Automatska - postaviš i zaboraviš
  • ✅ Pouzdana - nikad više izgubljeni projekti

Ako razvijaš software, radiš na važnim dokumentima, ili jednostavno ne želiš ponovno doživjeti taj osjećaj kada shvatiš da su sati ili dani rada nestali - ova aplikacija je za tebe.

🚀 Brzi početak

Instalacija i pokretanje

  1. Instaliraj potrebne module: pip install -r app/requirements.txt
  2. Pokreni aplikaciju: dvoklik na main.pyw ili python main.py

Kreiranje prvog job-a

  1. Klikni "➕ New Job"
  2. Unesi naziv job-a
  3. Odaberi tip (Simple ili Incremental)
  4. Postavi source i destination foldere
  5. Konfiguriraj raspored
  6. Označi "Active" za automatsko pokretanje
  7. Klikni "Save"

📁 Tipovi job-ova

Simple Job

Kako radi: Kreira potpunu kopiju source foldera s datumom i vremenom u nazivu.

Primjer: Dokumenti_20251002_143000

Dodatne opcije:

  • ZIP Compression: Kompresija kao ZIP datoteka
  • Exclude Patterns: Isključi datoteke (.git, node_modules, __pycache__)
  • Desktop Notifications: Obavijesti o završetku

Incremental Job

Kako radi: Radi s dva koncepta:

  • Sync Directory: Ažurna radna kopija (destination/folder_name/)
  • Snapshots: Periodičke arhivske kopije (destination/folder_INCREMENTAL_timestamp/)

Opcije:

  • Preserve deleted files: Zadrži obrisane datoteke
  • Create snapshots: Omogući snapshot-ove
  • Snapshot interval: Interval snapshot-ova
  • Exclude Patterns: Isključi datoteke
  • Desktop Notifications: Obavijesti o završetku
Važno: U Backup Files tabu vidiš samo snapshot-ove, ne sync directory. Sync directory je radna kopija, snapshot-ovi su arhivski backup-ovi.

🖥️ Korisničko sučelje

Tabovi

  • 📊 Dashboard: Statistike, uspješnost, sljedeći backup, aktivnost
  • 📋 Jobs: Lista job-ova s kontrolnim gumbovima
  • 📝 Logs: Logovi izvršavanja s filterima
  • 🗂️ Backup Files: Pregled i upravljanje backup datotekama

Kontrolni gumbovi (vidljivi samo u Jobs tabu)

  • ➕ New Job: Kreira novi job (uvijek vidljiv)
  • ✏️ Edit Selected: Uređuje odabrani job
  • 🗑️ Delete Job: Briše job
  • 📁 Open Destination: Otvara destination folder
  • ▶ Activate Job: Aktivira job
  • ⏸ Deactivate Job: Deaktivira job
  • ▶ Run Selected: Pokreće job odmah
Pametno sučelje: Kontrolni gumbovi su vidljivi samo kada je aktivan Jobs tab. Ovo čini sučelje čišćim i jasnijim.

🔽 System Tray

Aplikacija može raditi u pozadini. Minimiziranje šalje aplikaciju u system tray, X gumb zatvara aplikaciju.

  • Minimize: Šalje u tray
  • X gumb: Pita za potvrdu zatvaranja
  • Desni klik na tray ikonu: Show/Quit opcije

⚙️ Dodatne funkcionalnosti

Exclude Patterns

Isključi određene datoteke/foldere iz backup-a. Primjeri:

  • .git,node_modules,__pycache__
  • *.tmp,*.log,Thumbs.db
  • cache/*,temp/*

ZIP Compression

Dostupno samo za Simple job-ove. Kompresira backup kao ZIP datoteku za uštedu prostora.

Desktop Notifications

Prikazuje obavijesti kada job završi:

  • Uspjeh: Zelena obavijest s brojem datoteka
  • Greška: Crvena obavijest s opisom greške
  • Preskočeno: Žuta obavijest (nema promjena)

File Retention Policy

Automatsko brisanje starih backup-ova:

  • Keep Count: Zadrži zadnjih N backup-ova
  • Keep Days: Zadrži backup-ove N dana
  • Keep Size: Zadrži do N MB

🗃️ Baza podataka

Aplikacija koristi SQLite bazu (app/sync_backup.db) za:

  • Konfiguracije job-ova
  • Hash vrijednosti za detekciju promjena
  • Logove izvršavanja
  • Praćenje backup datoteka
  • Retention policy pravila

🛠️ Rješavanje problema

Job se ne pokreće

  • Provjeri je li job aktivan
  • Provjeri "Next Run" kolonu
  • Pogledaj logove za greške

Backup je prazan

  • Provjeri postoji li source folder
  • Provjeri dozvole za čitanje/pisanje
  • Provjeri exclude patterns

Aplikacija se ne pokreće

  • Provjeri jesu li instalirani svi moduli
  • Pokreni iz command prompt-a za greške
  • Provjeri Python verziju (3.7+)
